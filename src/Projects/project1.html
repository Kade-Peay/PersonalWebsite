Tic Tac Toe

This is a very simple command line tic tac toe game!

Composed of 171 lines of poorly written, weirdly formatted code; it was one of my first projects I did in python.

This has no A.I. for the enemy player, is only one player, and works by picking a number between 1 and 9 to choose where to put your 'X'

Of course if you'd like to learn more, you are always welcome to check out it's github page here: GitHub

How it works

It works in a simple way, start with a 2-dimensional array and print that to the console:

def boardinit():
	board = [['1', '2', '3'],
	['4', '5', '6'],
	['7', '8', '9']]
	return board

def printBoard(arr):
	for row in arr:
		print(row)

Now we get to the best bit of code, the function for the players turn. This is just a laundry list of if/elif statements. Each of these correspond to a certain position in the array. It checks if an 'X' or 'O' is in that position, if not, it places the 'X'. 

# Make this a choice between 1 and 9
def playerTurn():
    print("Choose Number between 1 and 9")
    choice = input("Enter number: ")

    if choice == '1':
        x = 0
        y = 0
    elif choice == '2':
        x = 0
        y = 1
    elif choice == '3':
        x = 0
        y = 2
    elif choice == '4':
        x = 1
        y = 0
    elif choice == '5':
        x = 1
        y = 1
    elif choice == '6':
        x = 1
        y = 2
    elif choice == '7':
        x = 2
        y = 0
    elif choice == '8':
        x = 2
        y = 1
    elif choice == '9':
        x = 2
        y = 2
    else:
        print('Invalid input')

    if board[x][y] == 'O' or board[x][y] == 'X':
        print("That spot is taken")
        playerTurn()
    else:
        board[x][y] = 'X'

Regardless now we come to the final important piece of the puzzle, and that is the main game "loop." I say "loop" because it actually doesn't loop. Once you win it doesn't start a new game. This would be an easy enough fix, and perhaps I'll do that later when I have more time and drive.

It's very straight forward: print who's turn, allow for player or computer input, check if anyone has won, print the board, and do it again. 

# I am going to use this to loop over the game
def game():
    print()
    print("Your Turn!")
    playerTurn()
    check()
    print("Computer's Turn!")
    compTurn()
    check()
    printBoard(board)
    print()
    game()
